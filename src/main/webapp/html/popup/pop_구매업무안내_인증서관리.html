인증서 사용 안내

※ 인증서 사용을 위해 공인인증서 발급 후 인증서 등록을 반드시 완료해야 합니다.

  • 공인인증서
    발급 안내

    전자서명법과 전자상거래 관련 법규에 근거하여, 거래당사자의 권익보호와 인터넷기반 전자상거래 질서를 유지하기 위하여 Open2U 사이트는 공인인증서를 통하여 거래당사자에 대한 전자인증과 거래 사실에 대하여 전자서명합니다.

  • 공인인증서
    종류 안내

    • 범용 공인인증서
      당사 사이트 뿐 아니라 타사 전자인증 사이트에서
      사용이 가능하며, 신청하신 회사에서 년단위로
      일정 비용이 소요 됩니다.
    • 용도제한용 공인인증서
      당사 사이트에서만 사용이 가능하며,
      SK텔레콤에서 비용을 부담하고 있습니다.
      SKT 용도제한용 인증서 발급
    • 사용자 여러분의 여건을 감안하여 필요한 것을 선택하여 발급 받으시기 바랍니다.
    • 기존 당사와 거래가 있어 이미 발급받은 인증서는 그대로 사용하실 수 있습니다.

  • 인증서 발급
    신청전 확인사항

    이미 국내 공인인증기관에서 범용 인증서를 구입하여 사용하고 있는 회사에서는 따로 구입하거나 신청하실 필요 없습니다.

    • 공인인증기관
      안내

      ※아래 공인인증 기관은 무순입니다.

      한국정보인증 한국무역정보통신 한국전자인증 코스콤
    • 신청절차

      인증서 종류 : 사업자용 범용 영문 인증서

      발급비용 : 인증기관에 문의 (위 인증기관명 클릭하여 확인하시기 바랍니다.)

    • 제출서류

      - 공인인증서 신청서 1부

      - 법인 또는 개인인감증명서 원본 1부

      - 사업자등록증 사본 1부

      - 대표자 또는 대리인 신분증 앞/뒤 사본 1부
