  • 대금지급바로 안내

    (舊 중소기업 Care프로그램)

    SK텔레콤은 동반성장Program의 일환으로 중소기업과의 거래 시 대금 조기지급를 위해 대금지급바로 (舊 중소기업 Care프로그램)를 운영하고 있으며, 대금지급바로에 가입 된 중소기업은 거래규모와 무관하게 대금을 조기 지급하고 있습니다.

    단, 중소기업이라 하더라도 신청을 하시지 않은 경우 일반조건으로 대금지급이 이루어 지고 있으니, 중소기업인 경우 꼭 신청하여 주시기 바랍니다.

  • 신청방법

    • 온라인신청
      SK텔레콤과 신규 거래를 하는 업체는 계좌등록
      완료 후 신청 가능
    • 중소기업 인정 기준
      중소기업확인서(중소기업현황정보시스템)
      제출 업체 한함

  • 온라인 신청방법

    • Step. 01
      http://open2u.sktelecom.com
      접속 (회원가입 필요)
    • Step. 02
      동반성장프로그램 → 금융지원 →
      대금지급바로 → 신청하기 Click
    • Step. 03
      중소기업확인서 첨부
